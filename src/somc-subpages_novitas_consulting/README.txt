== Description ==

Plugin that can be used as a Wordpress Widget and as a Wordpress Shortcode.

Fetches and display a list of sub pages on any page that the shortcode (below) is placed on.
Titles are truncated to 20 characters and displays a 'featured image' next to the title.

Each level can be sorted by ascending and descending order of page title by the user. Each level can be expanded and collapsed.

Customisable Widget allows a list of sub pages (of the current page) to be displayed in the sidebar.
Widget is managed from the admin backend and the user can define the widget title on the frontend.

Shortcode: [somc-subpages-novitas-consulting]


== Installation ==

1. Go to Plugins -> Add New -> Upload Plugin.
2. Click the "Browse" button and select the /dist/somc-subpages_novitas_consulting.zip file and click "Install Now" to install the plugin.
3. Click "Activate Now" - the plugin will now be ready to use.
